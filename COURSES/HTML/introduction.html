Introduction

  • HTML is the standard markup language for creating different web pages.
  • HTML stands for Hyper text Markup Language.It also used as full form of HTML.
  • It displays our Web pages in a structured form and tells the browser how to display the content.
  • HTML consists of a series of element.
  • HTML elements label pieces of content such as "This is a web page", "welcome to HTML course", etc.

Let us know with the help of above example:

  1. The < !DOCTYPE html > element defines that this document is an HTML document.
  2. The < html > element is the root element of an HTML page.
  3. The < head > element contains meta information about the HTML page.
  4. The < title > element contains title of the HTML page which is shown in the users browser's title bar or we can say in the page's tab.
  5. The < body > element contains all the visible contents of web page. We can say that it is a container which contain all data, such as heading, paragraph, images, vedios, tables, lists of different type, div blocks, etc.
  6. The < h1 > element defines the maximum size of heading or a large heading.
  7. The < p > element explains about paragraph.

Use of Web Browsers:

  • The purpose of Web Browsers (Chrome, Edge, Firefox) is to read the HTML document and display them correctly.
  • A browser does not display the HTML elements, but uses them to determine how to display the document.
